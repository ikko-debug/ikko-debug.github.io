常见排序算法概述表

排序算法 时间复杂度（平均） 时间复杂度（最坏） 空间复杂度 是否稳定 适用场景/特点
冒泡排序 $O(n^2)$ $O(n^2)$ $O(1)$ 是 简单，适合小数据量
选择排序 $O(n^2)$ $O(n^2)$ $O(1)$ 否 简单，交换次数少
插入排序 $O(n^2)$ $O(n^2)$ $O(1)$ 是 适合部分有序、小数据量
希尔排序 $O(n^{1.3})$ $O(n^2)$ $O(1)$ 否 改进插入排序，较快
快速排序 $O(n\log n)$ $O(n^2)$ $O(\log n)$ 否 实际应用最广泛
归并排序 $O(n\log n)$ $O(n\log n)$ $O(n)$ 是 稳定，适合大数据
堆排序 $O(n\log n)$ $O(n\log n)$ $O(1)$ 否 不稳定，适合大数据
计数排序 $O(n+k)$ $O(n+k)$ $O(k)$ 是 适合整数、范围小
桶排序 $O(n)$ $O(n)$ $O(n)$ 是 适合均匀分布数据
基数排序 $O(nk)$ $O(nk)$ $O(n+k)$ 是 适合数字、字符串排序

滑动窗口最大值

给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

最大堆

维护一个最大堆，（值，下标）。遍历数组进堆，然后弹出不符合滑动窗口的元素，堆顶即为当前窗口最大值。

单调队列

维护一个单调递减队列，遍历数组进队列，然后弹出不符合滑动窗口的元素，队首即为当前窗口最大值。若队列中当前最小值小于新加入值，则将其弹出，直到队列为空或队尾值大于新加入值。
